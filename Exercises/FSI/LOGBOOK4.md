## Task 2.6: The PATH Environment Variable and Set-UID Programs
#### Compile Set-UID program
```c
// task6.c
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
int main()
{
    printf("/home/seed/task6 (Set-UID program):\n");
    uid_t uid = getuid();
    uid_t euid = geteuid();
    printf("UID: %d\n", uid);
    printf("EUID: %d\n", euid);
    system("ls");
    return 0;
}
```
