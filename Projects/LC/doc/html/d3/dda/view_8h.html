Detailed Description

Definitions of all the drawing functions, i.e., the view of this MVC architecture.

The view is responsible for the sprites and for drawing every information in the screen using the video card driver, including the menu, the boards, the boats, the cursor and much more.

Authors
Inês Almeida, Guilherme Matos, João Ferreira, Maria Araújo
